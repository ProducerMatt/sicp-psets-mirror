Text of the problem set

Problem set in postscript format, for printing or viewing with a program like ghostview. (Note: This file was updated on April 6. The first version of the file located here was an older version that did not match what was distributed in class on March 30, although there aren't any serious differences. Sorry for the inconvenience.)

Problem set in HTML format, for Web browsing (including embedded links to the 6001 homework discussion forum).

Problem set in pdf format, for viewing with Adobe Acrobat (available from www.adobe.com).

Problem set in dvi format, for viewing a program like xdvi.

Code to load for this problem set

For GNU/Linux users
Gzipped tar file format. Put this tar file in the psets directory, go to that directory, and type
tar xzvf ps8.tar.gz.
If this works, you can remove the tar file.

For Windows95 and Windows NT users
Zip file format. Unzip this file in the into the directory you set up as your problem sets directory when you installed 6.001 Scheme (this will be a directory with the name psets under the directory you actually specified). The result should update the probsets file in the psets directory and also place in psets a directory called ps8 that contains two files: speech.scm and dict.scm.

Code to read for this problem set

When you load the problem set, the following code files (included in the tar and zip packages above) will be placed in Edwin buffers so that you can examine them easily. Here is another copy of this code, in a form that you can look at with a web browser:
  • speech.scm
  • dict.scm
